# Overwatch — Phone-First Successful Path

1. Build and certify the phone opening as an independent composition.
2. Keep the underlying film invisible and inert until the explicit start gesture.
3. Use one start gate and one director runtime.
4. Validate Safari/in-app browser safe areas before visual polish.
5. Never pin a CTA over a running desktop scene.
6. Never expose captions, navigation, or voice controls before the start gate.
7. Complete human start-to-finish testing on phone before adapting tablet or desktop.
