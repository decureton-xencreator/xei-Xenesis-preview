# Phone Gold Master V8

The phone premiere is a separate opening composition, not a compressed desktop scene.

## Mandatory opening contract

- One diamond.
- One time-aware greeting.
- One explanatory sentence.
- One Begin with Xen button.
- No captions before the tap.
- No voice before the tap.
- No navigation before the tap.
- No underlying scene motion before the tap.
- The button remains above browser chrome and safe-area obstruction.

## Post-start contract

The single director runtime owns narration, captions, scroll, scene progression, interaction pauses, restart, and navigation. Phone framing is validated independently before tablet or desktop adaptation.
